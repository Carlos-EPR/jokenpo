					Jokenpo Project


about:
The project is based on the development of a code in the Python language, which allows us to play a world-famous game, Jokenpo (Rock, paper or scissors), directly at our command prompt (terminal).


Prerequisites:
For the code to be used and the game to be able to be put into practice, you will need a Python language interpreter (PyCharm, VScode, among others)

How it works:
With your interpreter open, all we have to do is paste in the code and run it, when running the game it will ask the player to choose their option (rock, paper or scissors) based on the numbering of each.

Once the choice has been made, your opponent (computer) will also make a random choice.

With the choices made, the program will check the result, which could be a draw, defeat or victory.

At the end, the program gives you the option of continuing playing by pressing the "s" key or ending the program with the "n" key.
				


					 Projeto Jokenpo


sobre:
O projeto se baseia no desenvolvimento de um código na linguagem Python, qual permite que joguemos um jogo mundialmente conhecido, Jokenpo (Pedra, papel ou tesoura), diretamente no nosso prompt de comando (terminal).


Pré-requisitos:
Para que o código seja utilizado e o jogo consiga ser posto em prática, será necessário um interpretador da linguagem Python (PyCharm, VScode, entre outros)

Funcionamento:
Com seu interpretador aberto, só nos resta colar o código e executá-lo, ao executar o jogo pedira para que o jogador escolha sua opção (pedra, papel ou tesoura) baseado na numeração de cada.

Após a escolha ser realizada, o seu adversário (computador) também fara uma escolha de forma aleatória.

Com as escolhas feitas o programa verificara o resultado, podendo ser empate, derrota ou vitória.

Ao fim o programa te dá a opção de continuar jogando ao apertar a tecla "s" ou encerrar o programa com a tecla "n".
